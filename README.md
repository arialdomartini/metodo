Metodo
========


## Build

* Generate the Docker images [`libgit2`](libgit2/README.md) running:

```
./build.sh
```

    
* Modify and compile the C# project
* Enter the development environment (a Docker container based on the `libgit2` image) with:

```
./run.sh
```
    
* Execute metodo with

```
./metodo.sh [params]
```
